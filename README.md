A modern, fast, and responsive weather forecasting web application built from scratch. Focused on performance, high-quality UI/UX, and web development best practices. The visual design utilizes the **Glassmorphism** trend, populating real data through integration with public external APIs.

## ✨ Features

- **Global City Search:** Search for the weather conditions of any city around the world.
- **Smart Geolocation:** Automatically requests the user's location upon entry to provide local weather instantly.
- **Detailed Forecast:**
  - Current weather with 'feels like' temperature and detailed description.
  - Interactive carousel displaying hourly forecasts for the next 24 hours.
  - Extended forecast list covering the upcoming 7 days.
- **Advanced Metrics:** In-depth information including Humidity, UV Index, Visibility, Atmospheric Pressure, and Wind Speed.
- **Dynamic Themes:** The interface intelligently reacts and alters its color scheme to reflect current weather conditions (e.g., rain, night, sunny).
- **Accessible & Responsive:** A flexible layout designed to provide the same incredible experience on mobile devices and large desktop screens alike.

## 💻 Technologies & Tools

This project was built **without heavy frameworks** (like React or Angular) on purpose, to demonstrate a solid understanding of Web fundamentals and the JavaScript language:

- **HTML5** (Semantic)
- **CSS3** (Variables, Flexbox, CSS Grid, Animations, and *Glassmorphism*)
- **JavaScript ES6+** (Modularization, Async/Await, Promises, DOM Manipulation)
- **Vite** (Ultra-fast frontend tooling and bundler)
- **Lucide Icons** (Beautiful interface icons)
- **Open-Meteo API** & **BigDataCloud API** (Weather data and Geocoding consumption)

## 🚀 How to Run Locally

Follow these steps to run the project on your local machine:

### Prerequisites
You need to have [Node.js](https://nodejs.org) installed on your system.

### Installation

1. Clone this repository:
```bash
git clone https://github.com/diodanciguer/Weather-Dashboard.git
```

2. Navigate into the project folder:
```bash
cd Weather-Dashboard
```

3. Install Vite's development dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. The terminal will provide a local link (usually `http://localhost:5173/`). Open it in your browser and enjoy!

## 📡 About the APIs

This project stands out for being incredibly easy to clone and test, as it has no blockers or complex configuration requirements:

- [Open-Meteo](https://open-meteo.com/): An open-source weather API that **does not require API Keys**, making it perfect for public portfolio projects.
- [BigDataCloud (Free Reverse Geocoding)](https://www.bigdatacloud.com/): Used to convert exact coordinates (Lat/Lon) into a user-friendly city name.

## ✨ Funcionalidades

- **Busca Global de Cidades:** Pesquise pelas condições climáticas de qualquer cidade ao redor do mundo.
- **Geolocalização Inteligente:** Solicita a localização do usuário ao entrar e fornece o clima local automaticamente.
- **Previsão Detalhada:**
  - Clima atual com sensação térmica e descrição detalhada.
  - Carrossel com previsão de hora em hora para as próximas 24 horas.
  - Lista de previsão estendida para os próximos 7 dias.
- **Métricas Avançadas:** Informações detalhadas como Umidade, Índice UV, Visibilidade, Pressão Atmosférica e Velocidade do Vento.
- **Temas Dinâmicos:** A interface reage de maneira inteligente, alterando o padrão de cores para refletir climas como chuva, noite e sol.
- **Acessibilidade e Responsividade:** Layout flexível desenhado para proporcionar a mesma experiência incrível tanto em dispositivos móveis (Smartphones) quanto em telas grandes (Desktops).

## 💻 Tecnologias e Ferramentas Utilizadas

Este projeto foi construído **sem o uso de frameworks pesados** (como React ou Angular) propositalmente, para demonstrar sólido domínio sobre as bases da Web e da linguagem:

- **HTML5** (Semântico)
- **CSS3** (Variáveis, Flexbox, CSS Grid, Animações e *Glassmorphism*)
- **JavaScript ES6+** (Modularização, Funções Assíncronas / Promessas, Manipulação do DOM)
- **Vite** (Ferramenta de empacotamento ultrarrápida para o ambiente de desenvolvimento)
- **Lucide Icons** (Biblioteca de ícones de interface)
- **Open-Meteo API** & **BigDataCloud API** (Consumo de dados climáticos e de geolocalização)

## 🚀 Como executar este projeto localmente

Siga o passo a passo abaixo para rodar o projeto em sua máquina:

### Pré-requisitos
Você vai precisar ter o [Node.js](https://nodejs.org) instalado na sua máquina.

```bash
git clone https://github.com/diodanciguer/Weather-Dashboard.git
```

2. Acesse a pasta do projeto:

```bash
cd Weather-Dashboard
```

3. Instale as dependências de desenvolvimento do Vite:

## 📡 Sobre as APIs Integradas

Este projeto se destaca por ser muito fácil de ser clonado e testado por não possuir bloqueios ou exigência de configurações difíceis:

- [Open-Meteo](https://open-meteo.com/): Uma API de clima e previsão de tempo em código aberto que **não requer Chaves de API (API Keys)** para projetos de portfólio.
- [BigDataCloud (Free Reverse Geocoding)](https://www.bigdatacloud.com/): Utilizada de forma auxiliar para transformar as coordenadas exatas da sua localização (Lat/Lon) no nome amigável da sua cidade.
